When starting to build a [value stream], ideally we would start with the Motivation Map.

**Stakeholder** - This is usually the person receiving the value at the end of the Value Stream
- **Triggers** - There can be many ways that a value stream is triggered. Sometimes it’s a direct event by the stakeholder like walking into a retail store or going to a website. Other times, it’s indirect due to a [[business capability instance]] triggering the value stream as part of a [[Value Stream Network]].
- **Stages** - A value stream usually has from 4-7 value stages. To be defined as a value stage, there needs to be some measure of value from the perspective of the value stream stakeholder associated with the stage.
- Value - value is not only associated with the value stream stage, but also the value stream overall.  Sometimes value will also be aligned to a strategic goal or a measure on the [[Balanced Scorecard - Translating Strategy to Action]].
