Do your value stream models start with activities, or intention. It is usually best to start with the end in mind.

While many might equate money for value, value is usually in the eye of the beholder. One of the key principles of a [[Value Streams|value stream]] is to start by identifing the value from the triggering stakeholder's perspective. That said, there is value for the roles that enable the values stream's business capabilities as well as for regulators and other external stakeholders.
