A data product is a self-contained, reusable data asset designed to deliver value to data consumers. A data product goes beyond raw data to include the context, quality controls, and documentation that make the data actually usable.

Data products treat data like a product you'd ship to customers - it should be discoverable, understandable, trustworthy, and fit for purpose. Not just "here's some data" but "here's data you can actually use to make decisions."
