An end-to-end sequence of activities that delivers value to a stakeholder. Value streams model how work flows through an organisation from a triggering event to a valuable outcome - from the perspective of the customer or stakeholder receiving that value.

Value streams are decomposed into value stream stages, each of which is enabled by [[Business Capability|business capabilities]]. Unlike process models (which focus on how work is done), value streams focus on what value is created and for whom.
