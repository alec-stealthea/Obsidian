---

A technology approach that provides a unified view of data from multiple sources without physically moving or copying the data. Instead of replicating data into a central repository, data virtualisation creates a logical layer that queries source systems on demand.

The advantage is near real-time data access and reduced storage costs. The trade-off is query performance and dependency on source system availability. Useful when data latency requirements are tight, but not a replacement for a proper data lake when historical analysis matters.
