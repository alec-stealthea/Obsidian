An RDF graph consists of a mesh of nodes connected by edges. A molecule is a graph (and RDF graphs and molecules actually have a lot in common). A relational, Codd-based database which consists of tables, rows, and columns is a graph. So is an XML or JSON document. In general, there are very few data structures that cannot be represented as a graph.
