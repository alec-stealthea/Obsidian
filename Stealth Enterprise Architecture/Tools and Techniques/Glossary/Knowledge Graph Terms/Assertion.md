In semantic terms, an assertion is a statement (or set of statements) that an authority is treating as being true for purposes of discussion. Note that such an assertions may not in fact describe empirical reality. An assertion would be a set of statements such as "A unicorn is a horse with a single horn emerging from its forehead. Binky is a horse with a single horn emerging from its forehead, therefore, Binky is a unicorn." If you included the expression "(hypothetical)" before unicorn and Binky, the assertion would be true regardless of the fact that unicorns don't exists in reality (arguably - which is the distinction between a rationalist and an empiricist).
