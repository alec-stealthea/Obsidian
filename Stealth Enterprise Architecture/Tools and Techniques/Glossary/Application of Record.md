The authoritative application where a specific type of data is first created and maintained. When multiple applications hold copies of the same information (customer address, product price, employee details), the application of record is the one whose data is considered authoritative.

Not to be confused with "Authoritative Application" which refers to where data is trusted for a specific context. The application of record is where data is born; the authoritative application is where it's trusted. Sometimes they're the same, often they're not.
