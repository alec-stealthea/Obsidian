Any collection of data that has value to an organisation. Data assets are the building blocks from which data products are created. While a data asset might be a raw table or dataset, a data product packages that asset with metadata, quality rules, and access patterns that make it consumable.

The distinction matters: you might have hundreds of data assets, but only the ones curated for reuse become data products.
