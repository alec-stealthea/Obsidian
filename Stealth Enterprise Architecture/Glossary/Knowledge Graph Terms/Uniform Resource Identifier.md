ThA **Uniform Resource Identifier (URI)** is a unique string used to identify a resource on the internet or within a system. It can represent a location (like a web page via a URL), a name, or an identifier for abstract or physical entities. URIs ensure resources can be referenced consistently and include two main types: **Uniform Resource Locator (URL)**, which specifies the resource’s address, and **Uniform Resource Name (URN)**, which specifies a resource by name, independent of location.
